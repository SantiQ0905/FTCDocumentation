# RoadRunner - Path Following

## ¿Qué es RoadRunner?

Librería de seguimiento de trayectorias usando splines. Muy popular en FTC, especialmente para autonomous avanzados.

## Instalación

```gradle
repositories {
    maven { url = 'https://maven.brott.dev/' }
}

dependencies {
    implementation 'com.acmerobotics.roadrunner:core:0.5.6'
}
```
