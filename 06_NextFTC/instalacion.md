# Instalación de NextFTC

## Gradle Setup

```gradle
repositories {
    maven { url = 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.NextFTC:NextFTC:0.6.1'
}
```

## Verificación

```java
import org.nextftc.nextftc.NextFTC;

// Si compila, está correctamente instalado
```
